# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.1.0] - 2024-12-20
### Added
- Support for retransforming already loaded classes

## [2.0.0] - 2023-10-20
### Added
- Java 21 support

### Changed
- Target Java version from Java 8 to Java 11
- Unit tests from JUnit 4 to JUnit Jupiter

## [1.4.0] - 2022-03-02
### Added
- Java 17 & 18 support

## [1.3.0] - 2020-09-30
### Added
- Java 15 support

## [1.2.1] - 2019-03-13
### Fixed
- `java.time.Clock$SystemClock.instant()` wasn't transformed properly in Java 9+.

## [1.2.0] - 2019-01-25
### Added
- Java 11 support

## [1.1.0] - 2018-05-03
### Added
- Java 9 & 10 support

## [1.0.0] - 2016-11-07
### Added
- The initial TimeTransformer agent
